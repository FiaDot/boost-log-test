boost-log-test
==============

Boost.Log V2 test code



## 테스트 항목

로그에 대해 화면과 파일저장 

파일당 용량은 최대 10MB -> 넘어가면 파일명에 %N% 증가

log 폴더에 최대 용량은 100MB -> 넘어가면 LRU로 삭제
